BulkPilot premium static website

Files:
- index.html
- styles.css
- script.js
- assets/

How to use:
1. Upload this folder to a new Render Static Site or push to GitHub.
2. Set the publish directory to the project root.
3. Replace # links with your real Shopify install link, privacy policy, terms, and support links.
4. Replace placeholder screenshots in /assets with your real screenshots if you want.

Suggested setup:
- bulkpilot.net -> this static site
- app.bulkpilot.net -> your existing Shopify app
